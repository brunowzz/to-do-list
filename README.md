## :page_facing_up: Explicação

Foi um projeto desenvolvido no DevClub, e funciona como uma lista de tarefas a serem adicionadas no seu dia.


## 📁 Páginas

O site tem no total 2 páginas, sendo elas:
<ul>
  <li> Home: primeira e única página que faz o registro das atividades do seu dia.  </li> 
</ul>

## :closed_book: Como usar? ##

Antes de iniciar, você precisa ter [Git](https://git-scm.com) e [Node](https://nodejs.org/en/) instalados.

```bash
# Clone this project
$ git clone LINK DO SEU REPOSITÓRIO
# Access
$ cd NOME DA SUA PASTA
# Install dependencies
$ yarn or npm 
# Run the project
$ yarn start or npm start 
# The server will initialize in the <http://localhost:3000>
```
